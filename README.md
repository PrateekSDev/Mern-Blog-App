# MERN Blog App

A simple full-stack blog application.

## Tech Stack
- **Frontend**: React (Vite), Tailwind CSS
- **Backend**: Node.js, Express, MongoDB
- **Auth**: JWT (Access & Refresh tokens)

## Features
- User Registration & Login
- Create, Read, Update, and Delete blog posts
- Markdown support for post content
- Responsive design for mobile and desktop

## Setup Instructions

### Backend
1. Go to `server` folder: `cd server`
2. Install dependencies: `npm install`
3. Create `.env` file from `.env.example` and add your MongoDB URI and JWT secrets.
4. Start server: `npm run dev`

### Frontend
1. Go to `client` folder: `cd client`
2. Install dependencies: `npm install`
3. Create `.env` file from `.env.example`.
4. Start client: `npm run dev`

## API Routes
- `/api/auth`: register, login, refresh, logout
- `/api/posts`: CRUD operations for blog posts
